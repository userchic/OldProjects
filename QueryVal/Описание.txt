Внимание!  пример использует  MS .NET 2.0 !

Это пример использования Веб-сервиса для получения ежедневных/ежемесячных данных курсов валют.
http://www.cbr.ru/DailyInfoWebServ/DailyInfo.asmx

Вся информация через веб-сервисы Банка России передается свободно, авторизации не требуется.


Проект - MS Visual Studio 2008 C#


Для того что бы использовать программу через ваш прокси сервер - в файле QueryValuta.exe.config нужно изменить в секции:
 <defaultProxy enabled="false" useDefaultCredentials="true">
      <proxy proxyaddress="http://YourProxyServer:3128" />
  </defaultProxy>

на:
enabled="true" 
proxyaddress - на адрес и порт вашего прокси сервера

если ваш прокси - сервер требует авторизации отличной от авторизации в IE:
1. укажите в файле QueryValuta.exe.config атрибут useDefaultCredentials="false". 
2. После запуска программы, нажмите кнопку соединения с сервером, при этом заполните форму ProxyParam, где укажите параметры соединения.
При этом будет создан файл QWconf.dat с данными. При следующем обращении форма не выдается и используются ранее указанные данные. 
3. Если вы ошиблись с параметрами подключения, то сотрите файл QWconf.dat, и повторите пункт 2.

Внимание, файл QWconf.dat не зашифрован.
